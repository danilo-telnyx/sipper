## Interactive Elements

### Click/Tap Targets
- Date range dropdown вҶ’ Change data timeframe
- Auto-refresh icon вҶ’ Toggle polling on/off
- Manual refresh icon вҶ’ Fetch latest data now
- Export dropdown вҶ’ Download JSON or CSV
- Run Test button вҶ’ Navigate to test runner
- Add Credential button вҶ’ Navigate to credentials
- Stats card (credentials) вҶ’ Navigate to credentials
- Test row in RecentTests вҶ’ Navigate to test details
- View all tests link вҶ’ Navigate to full test results
- Sort buttons вҶ’ Toggle sort order
- Filter dropdowns вҶ’ Filter visible tests
- Search input вҶ’ Real-time search

### Hover States
- All cards have shadow on hover
- Test rows highlight on hover
- Timeline events highlight on hover
- Buttons show hover effects
- Links show underline on hover

### Loading States
- Skeleton loaders for:
  - Stats cards (4x)
  - Recent tests (5x rows)
  - Activity timeline (5x events)
  - Charts (animated pulse)
- Spinner on refresh button when fetching
- Spinning refresh icon when auto-refresh active

### Color Coding
- **Green (#22C55E):** Success, excellent, passing
- **Red (#EF4444):** Failure, critical, failed
- **Yellow (#EAB308):** Warning, fair, pending
- **Blue (#3B82F6):** Info, neutral, primary actions
- **Gray (#6B7280):** Muted, secondary, inactive

### Animations
- Chart transitions (300ms ease)
- Skeleton pulse (2s infinite)
- Hover scale on icons
- Fade in for loaded content
- Smooth scroll to sections

## Performance Considerations

- **Code Splitting:** Dashboard components are separate bundle
- **Lazy Loading:** Charts only render when visible
- **Memoization:** Expensive calculations cached with useMemo
- **Debouncing:** Search input debounced (300ms)
- **Pagination:** Ready for large datasets
- **Virtualization:** Can add if table grows large (react-window)
- **Image Optimization:** Icons are SVG (no raster images)
- **Tree Shaking:** Only used Chart.js components imported
